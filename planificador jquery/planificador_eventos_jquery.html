Planificador de Eventos

Herramienta sencilla para añadir un evento a nuestra agenda: Nombre del Evento, Fecha del Evento y Listado de asistentes. Permite añadir, eliminar y reordenar los asistentes, y contiene dos botones para confirmar o cancelar la operación.

A tener en cuenta

El documento contiene todo el HTML y CSS necesario para su desarrollo. Sólo tienes que añadir el código Jquery requerido. Aunque si lo crees conveniente puedes cambiar el HTML o CSS a tu gusto.
En el head del documento están enlazados los archivos necesarios para que este documento funcione correctamente, son tres: jquery.js, jquery-ui.js, jquery-ui.css, no cambies los nombres ni las rutas de estos archivos.

Elemento datepicker

Para seleccionar la fecha del evento usa el componente de jquery datepicker. Sólo tienes que asociarlo al input donde quieres recoger la fecha con la siguiente expresión:
$( "#id_del_input" ).datepicker($.datepicker.regional[ "es" ]);

Elementos Reordenables

Para poder hacer reordenables los elementos de la lista usamos otro de los componentes de jquery-ui. En este caso es el método sortable(). Para hacer que los elementos li de un contenedor ol (por ejemplo) sean ordenables, sólo tenemos que indicarle a ese contenedor ol que sea sortable(). Algo así:
$("#mi_lista_ol").sortable();
