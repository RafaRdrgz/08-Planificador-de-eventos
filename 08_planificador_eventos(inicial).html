Planificador de Eventos

Herramienta sencilla para añadir un evento a nuestra agenda: Nombre del Evento, Fecha del Evento y Listado de asistentes. Permite añadir, eliminar y reordenar los asistentes, y contiene dos botones para confirmar o cancelar la operación.

A tener en cuenta

El documento contiene todo el HTML y CSS necesario para ver una muestra de lo que hay que hacer. Pero puedes modificarlo como quieras

Elemento datepicker

Para seleccionar la fecha del evento podemos usar un input de tipo date o buscar un componente de React datepicker.

Elementos Reordenables

Para poder hacer reordenables los elementos de la lista usamos otro de los componentes desiponibleds en las librerias de React. Hay varias disponibles, solo tienes que hacer una búsqueda de componentes Drag and Drop de React.

Crear Evento

Pulsa en el botón para añadir los datos del evento: Nombre, fecha y participantes.
